Open Source at Heart

فناوری‌ها

Sarbala builds games entirely with open-source tools — reviewed, trusted, and always up to date.

سربالا برای ساخت بازی‌ها از فناوری‌ها و پروژه‌های متن‌باز استفاده می‌کنه.

این پروژه‌ها کارامدترین ابزار جهان برای رسیدن به مقصدند. هر خط کد در پروژه‌های متن‌باز توسط میلیون‌ها نفر بررسی می‌شه تا هیچ شکی در بهینه بودن اونها باقی نمونه. پروژه‌های متن‌باز مدام با آخرین فناوری روز به‌روزرسانی می‌شن.

Open-source software is developed in a collaborative public manner — code reviewed by millions worldwide, always efficient and trustworthy.

Why Sarbala?

چرا سربالا؟

Sarbala means "uphill" in Farsi. The path of game making was new and hard for us, but the more we climbed, the more we loved it.

چون این مسیر برای ما کاملاً جدید و سخت بود، ولی هرچه بیشتر پیش می‌رفتیم بیشتر عاشقش می‌شدیم — سربالا.

🥼
